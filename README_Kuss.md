# PandExo used in Kuss et al. (2026)

This repository contains the PandExo setup, input files, and analysis scripts used for the simulations presented in:

> Kuss et al. (2026), "Climates of temperate rocky planets with He-dominated atmospheres"

## Relevant files

### Input spectra
- pandexo_lhs_spectrum_new1.txt
- pandexo_lhs_spectrum_new2.txt
- pandexo_lhs_spectrum_new3.txt

### Analysis scripts
- analyze_data.py
- PAP_analyze_data_obs_data.py

### Simulation setup
- pandexo_generate.py

### Additional files
- Damiano_JWST.rtf (for plots)
- patch_numpy.py (compatibility)

## Software versions
The simulations were performed using:

- PandExo v3.1.1
- Pandeia Engine v3.0.0
